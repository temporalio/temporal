NAME:
   tdbg shard list-tasks - List tasks for given shard ID and task category

USAGE:
   tdbg shard list-tasks [command options]

OPTIONS:
   --more                     List more pages, default is to list one page of default page size 10 (default: false)
   --pagesize value           Result page size (default: 1000)
   --shard-id value           The ID of the shard (default: 0)
   --task-category value      Task category: visibility, replication, memory-timer, outbound, transfer, timer
   --min-task-id value        Inclusive min taskID. Optional for transfer, replication, visibility tasks. Can't be specified for timer task (default: 0)
   --max-task-id value        Exclusive max taskID. Required for transfer, replication, visibility tasks. Can't be specified for timer task (default: 0)
   --min-visibility-ts value  Inclusive min task fire timestamp. Optional for timer task. Can't be specified for transfer, replication, visibility tasks. Supported formats are '2006-01-02T15:04:05+07:00', raw UnixNano and time range (N<duration>), where 0 < N < 1000000 and duration (full-notation/short-notation) can be second/s, minute/m, hour/h, day/d, week/w, month/M or year/y. For example, '15minute' or '15m' implies last 15 minutes.
   --max-visibility-ts value  Exclusive max task fire timestamp. Required for timer task. Can't be specified for transfer, replication, visibility tasks. Supported formats are '2006-01-02T15:04:05+07:00', raw UnixNano and time range (N<duration>), where 0 < N < 1000000 and duration (full-notation/short-notation) can be second/s, minute/m, hour/h, day/d, week/w, month/M or year/y. For example, '15minute' or '15m' implies last 15 minutes.
   --print-json               Print in raw json format (default: true)
   --help, -h                 show help
