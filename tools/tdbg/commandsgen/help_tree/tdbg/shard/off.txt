NAME:
   tdbg shard - Run admin operation on specific shard

USAGE:
   tdbg shard command [command options]

COMMANDS:
   describe, d        Describe shard by ID
   list-tasks         List tasks for given shard ID and task category
   close-shard        close a shard given a shard id
   remove-task, rmtk  remove a task based on shardId, task category, taskId, and task visibility timestamp
   help, h            Shows a list of commands or help for one command

OPTIONS:
   --help, -h  show help
