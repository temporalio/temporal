NAME:
   tdbg dlq list - List all DLQs, only for v2

USAGE:
   tdbg dlq list [command options]

OPTIONS:
   --output-filename value  Output file to write to, if not provided output is written to stdout
   --pagesize value         Page size to use when listing queues from the DB (default: 1000)
   --print-json             Print in raw json format (default: false)
   --help, -h               show help
