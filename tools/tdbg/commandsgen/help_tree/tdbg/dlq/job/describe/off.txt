NAME:
   tdbg dlq job describe - Get details of the DLQ job with provided job token

USAGE:
   tdbg dlq job describe [command options]

DESCRIPTION:
   This command will get details of the DLQ job with provided job token if using v2

OPTIONS:
   --job-token value  Token of the DLQ job. This token will be printed in the output of merge and purge commands
   --help, -h         show help
